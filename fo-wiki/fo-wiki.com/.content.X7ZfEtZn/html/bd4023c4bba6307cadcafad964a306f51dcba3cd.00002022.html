How to get started
  • 1. Register on this site! You will need your game login. Be very careful about lower/upper case!
  • 2. Download this file.
  • 3. Extract it to a seperate folder!
  • 4. Run the update.exe and wait until it says "update complete".
  • 5. Start the logger.exe and enter your login details for this site!
  • 6. There should be a help window, follow instructions, you're almost done!
  • 7. Once everything is working fine, the symbol (bottom right) should turn green.
  • Note: The Logger only works under Windows!
If you have any problems, please post on the forum, send a bug report or msg me with a pm ingame/forum! Same goes if you have suggestions! This will help to improve things!


The Logger is a program that has many functions to support your game play. It was originally developed to help balancing the game. However, its functionality has grown and currently it supports:
  • display exp tnl, exp per kill, kills tnl
  • one-click item upload to fo-wiki.com
  • Autoupdater
  • Minimap
  • External Chat
  • Keeping track of you game play and character development.
  • Visualizing this development
  • Logging all chat and saving it to "chatlog.txt"
  • Converting your item and skill icons to *.jpg (for item and skill db)
  • Read NPC chat text (when window is open) and save it to npc_text.txt
  • Record map screen shots, align them and combine them to one big map
  • Upload your character so that you can compare it with other players
  • Recording game play videos (example)
External Chat
  • use an interface to chat, you can leave the fo window somewhere in the background (dont minimize it though)
  • whispers are sticky, so just select all the ppl you talked to so far from a drop down menu (and delete them hitting backspace)
  • large chat window, so you can still see whats going on, even when the chat moves fast

Character Upload
  • just click "Upload char" while the logger is "green"
  • note that the logger will need to access the Internet at this point!
  • the ranking can be found here.

Map Creation
  • click "Start map snapshot" and start walking through the map
  • click "Stop map snapshot"
  • start FineAlign.exe and click "Start". wait till finished.
  • start MapIt.exe and click "create_map". wait till finished.
  • the map should now be in your the same folder called "map.bmp", be aware that previous "map.bmp" will be overwritten!
  • note: delete the screen shots from the map folder if you wanna start over!
show
Game Log
  • The logger only "records" your game log as a txt file (log.txt). To get a visualization use the LogtoBmp.exe
show
Video recording
  • The logger only "records" your video as screen shots (/vid folder). So you'll need to convert those using the BmpToVideo.exe. Because BmpToVideo is a evaluation version there might be problems (Evaluation Time Over) with large videos. If so, use virtualdub to convert the bitmaps to avi. For the frame rate use the value that you find in /vid/fps.txt. A nice tutorial how to do it can be found here
  • The length of your video is limited by your free ram
  • Keep in mind that the result video is not compressed. You'll have to use a tool of your choice to compress it (i can recommend virtualdub). For codecs just search for a codec pack ("codecs virtual dub") @ google.
    In the example I'm using the Xvid MPEG-4 Codec. To select the Codec Click Video->Compression in Virtualdub. Then click File->Save as AVI
